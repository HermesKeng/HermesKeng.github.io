Embodied Weather

February, 2019 - January, 2020

Embodied Weather is a multi-sensory virtual reality (VR) experience to teach the public's awarness of different extreme weathers and promote the everyday low carbon behaviors. In this project, I worked with Hong Kong Observatory to design this project.

KusoCam

April, 2018 - May, 2018

Kusocam was an assignment in computer vision course. The main goal of the project was to simulate make-up effects in an iOS app. Therefore, I created the app from prototype to implementation with the AR-kit framework.

Image Processor

March, 2018 - April, 2018

Image Processor is web application. This project was an assignment in computer vision class. In this project, I demonstrated different algorithms of edge detection by using Javascript and HTML canvas to render a processed image

LoRa Logistic Service

July, 2016 - July, 2017

LoRa Logistic Service is an application for a self-driving car. Users can use the mobile app to register the service to deliver goods to a destination on campus. The purpose is to explore the possibility to apply LoRa technology in our daily lives.

Scotland Yard

December, 2016 - February, 2017

Scotland Yard is a famous board game in the world. I take this board game as reference and change some rules with the goal to creating an online interactive game for a thirty-people workshop.
